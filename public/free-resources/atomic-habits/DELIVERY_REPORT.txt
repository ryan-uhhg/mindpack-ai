내용:
- 제목: Atomic Habits — 작은 습관, 큰 변화
- 저자: James Clear
- 핵심 개념 3가지:
  ✓ Identity → Process → Outcome (정체성-프로세스-결과 루프)
  ✓ 각 단계별 상세 설명과 실제 사례
  ✓ 시각적 루프 다이어그램 포함

- 4 Laws of Behavior Change:
  1. 명확성 (Make It Obvious)
     - 습관의 신호를 명백하게 만들기
     - 환경 디자인과 습관 스택
  
  2. 매력성 (Make It Attractive)
     - 습관을 원하게 만들기
     - 보상과 즉각적 피드백
  
  3. 용이성 (Make It Easy)
     - 행동의 마찰력 줄이기
     - 2분 규칙 적용
  
  4. 만족감 (Make It Satisfying)
     - 즉각적 보상으로 강화하기
     - 습관 추적 방법

특징:
- 미니멀한 디자인 ($150M 에이전시 수준)
- Noto Sans KR 폰트 사용
- 흰색 배경으로 프린트 최적화
- 페이지 분할 방지 CSS 적용
- 시각적 강조 (박스, 색상) 포함
