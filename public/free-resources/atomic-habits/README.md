### 2. 02_tracker_template.pdf (250KB)
**30일 습관 추적 템플릿 - 3페이지**

내용:
- 시작/종료 날짜 입력란
- 사용자명 입력란
- 4가지 습관 목표 기입란
- 30일 일일 체크박스 (4주 + 2일)
- 주간 누적 점수 기록란 (28점 만점 × 4주)
- 최종 30일 누적 점수 (120점 만점)
- 팁 및 안내 문구

디자인:
- 폰트: Noto Sans KR
- 배경: 흰색
- 그리드: 시각적 명확성을 위한 구획
- 최적화: A4 프린트 최적화, 페이지 분할

### 3. 03_video_summary.md (7.3KB)
**영상 요약 스크립트 - Markdown 형식**

내용:
- 오프닝 (30초)
- 핵심 1: 정체성이 모든 것을 결정한다 (2분 30초)
  - 결과 중심 vs 정체성 중심 비교
  - 명상 습관 형성 사례
- 핵심 2: 행동 변화의 4가지 법칙 (2분 30초)
  - 명확성, 매력성, 용이성, 만족감
  - 각 법칙별 구체적 실행 방법
- 핵심 3: 환경이 습관을 결정한다 (1분 30초)
  - 아이스크림 회사 직원 사례
  - 피트니스 센터 거울 설치 사례
  - 환경 설계 3가지 규칙
- 클로징: 실전 코칭 메시지 (1분)

특징:
- 총 8:30 ~ 10분 분량
- 흐름: 이론 → 사례 → 실제 적용
- 텍스트 오버레이 가이드 포함
